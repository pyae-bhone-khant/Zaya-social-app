## 📝 Project Description

Social Hub is a feature-rich social media application that provides users with a modern and intuitive platform to connect, share, and engage with others. Built with cutting-edge web technologies, it offers a seamless experience with real-time interactions, beautiful UI design, and robust functionality.

---

## 🚀 Key Features

### 🔐 **Authentication & User Management**
- ✅ User registration and login system
- ✅ Secure JWT-based authentication
- ✅ Profile management and customization
- ✅ Follow/unfollow functionality
- ✅ User activity tracking

### 📱 **Social Features**
- ✅ Create, edit, and delete posts
- ✅ Real-time commenting system
- ✅ Like and unlike posts
- ✅ User interaction tracking
- ✅ Timeline/feed functionality

### 🎨 **Modern UI/UX**
- ✅ Beautiful gradient-based design
- ✅ Dark/Light theme support
- ✅ Glassmorphic card designs
- ✅ Smooth animations and transitions
- ✅ Fully responsive layout
- ✅ Professional typography with Inter font

### ⚡ **Performance & Architecture**
- ✅ Fast development with Vite
- ✅ Efficient state management with React Query
- ✅ Component-based architecture
- ✅ Optimized bundle size
- ✅ Code splitting and lazy loading

## 🚀 Getting Started

### **Prerequisites**
- Node.js (v16.0.0 or higher)
- npm or yarn package manager
- Modern web browser

1. **Install dependencies**

   npm install

2. **Start development server**

   npm run dev

### **Build for Production**

```bash
# Build the application
npm run build

# Preview the production build
npm run preview
```


## 🔧 Configuration

### **Environment Variables**
Create a `.env` file in the root directory:

```env
VITE_API_BASE_URL=http://localhost:8800
VITE_APP_NAME=Social Hub
```

### **Theme Customization**
The theme system is fully customizable in `src/AppProvider.jsx`:

```javascript
// Modify colors, typography, and component styles
const theme = createTheme({
  palette: {
    primary: {
      main: '#6366f1', // Your brand color
    },
    // ... other theme customizations
  },
});
```
